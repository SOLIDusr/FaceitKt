Getting Started


FaceitKt is a simple yet powerful tool that allows you to easily interact with the Faceit API. It is written in 100% Kotlin and is fully open-sourced.

It uses Gson:2.10.1 and OkHttp:4.12.0 to make requests to the Faceit API. Current version of FaceitKt is 1.0.0.
You can always check the latest version of FaceitKt on GitHub(*clickable*).

Report any issues to our github issue tracker

Main Components

Important parts of code to know:

FaceitClient

FaceitClient is the main class that you will use to interact with the Faceit API. It contains all the endpoints and methods to interact with the Faceit API.

Private value "apiHeader"

Private value "apiHeader" is a map of headers that will be used to make requests to the Faceit API.
You can add your own FaceitAPI key when you initialize the FaceitClient object, but my key already here if you don't have one.

Operator fun Any?.get

Operator fun Any?.get is a fun thing that allows you to access the value of the object if code's unsure weather it's a map or list or whatever.
It's not recommended to use it, but it's a fun thing to know.

fun getRequest(url: String)

fun getRequest(url: String) is a function that makes a GET request to the Faceit API.
It takes in a string of the url to make the request to and returns a Response object.
