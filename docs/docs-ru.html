Начало работы


FaceitKt - это простой и мощный инструмент, который позволяет вам легко взаимодействовать с Faceit API. Он написан на 100% Kotlin и полностью открытый исходный код.

Он использует Gson:2.10.1 и OkHttp:4.12.0 для отправки запросов на Faceit API. Текущая версия FaceitKt - 1.0.0.
Вы всегда можете проверить последнюю версию FaceitKt на GitHub(*кликабельно*).

Отправьте любые проблемы на наш issue tracker

Основные компоненты

Важные части кода, которые нужно знать:

FaceitClient

FaceitClient — это основной класс, который вы будете использовать для взаимодействия с Faceit API. Он содержит все конечные точки и методы для работы с Faceit API.

Приватное значение "apiHeader"

Приватное значение "apiHeader" — это карта заголовков, которые будут использоваться для отправки запросов к Faceit API.
Вы можете добавить свой собственный ключ FaceitAPI при инициализации объекта FaceitClient, но мой ключ уже здесь, если у вас его нет.

Оператор fun Any?.get

Оператор fun Any?.get — это забавная функция, которая позволяет получить значение объекта, если код не уверен, является ли он map, списком или чем-то еще.
Не рекомендуется использовать его, но это забавная вещь, о которой стоит знать.

fun getRequest(url: String)

fun getRequest(url: String) — это функция, которая отправляет GET-запрос к Faceit API.
Она принимает строку с URL для запроса и возвращает объект Response.
